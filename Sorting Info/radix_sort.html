Radix Sort is a Sorting Algorithm capable of sorting huge arrays whose elements Base is as big as 256.
Let's say we have an array of 5 elements with the biggest being 999



First we need to make a Dictionary that contains all the numbers from 0 - 9



Then we look at the last digit of every number and then assing the value we found accordingly







And now we drop every element to a new array



2nd Iteration

We know repeat the same proccess but now we are looking at the second from the end number



If the number does not have enough digits we set their found value to 0







And now we drop every element to a new array



3rd Iteration

And now we repeat the same proccess again for the last time.
