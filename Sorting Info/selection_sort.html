Sorts / Selection Sort Info

The selection sort algorithm sorts an array by repeatedly finding the minimum element
(considering ascending order) from the unsorted part and putting it at the beginning.

The algorithm maintains two subarrays in a given array.



In every iteration of the selection sort, the minimum element (considering ascending order)
from the unsorted subarray is picked and moved to the sorted subarray.
Let's consider the following array as an example:

[ 47,23,59,15,34 ]


For the first position in the sorted array, the whole array is traversed from
index 0 to 4 sequentially. The first position where 47 is stored presently,
after traversing whole array it is clear that 15 is the lowest value.



Thus, replace 47 with 15. After one iteration 15, which happens to be
the least value in the array, tends to appear in the first position of the sorted array.



Now the iterations will be without 15, because it is already in the sorted array (sorted),[main]


For the second position, where 23 is present,
again traverse the rest of the array in a sequential manner.



There isn't any smaller element than 23 so it's
the 2nd element in the sorted array.





For the third position, where 59 is present, again traverse the rest
of the array and find the third least value present in the unsorted array.
The third least value is 34 so replace it with 59.



Now, for the last 2 position, similarly, 47 is present,
traverse the rest of the array and find the fourth least element in the array.
The 4th least value is 47 so it remains and so does 59. Our sorted array is:

Iteration: 0
